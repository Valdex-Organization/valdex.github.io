# Valdex Website

Welcome to the **official Valdex website repository**!  

Valdex is a **tech + creative organization** focused on delivering custom gaming experiences, tools, and mods for developers and gamers alike. This repository contains the source code for our website, which serves as a showcase for all Valdex projects and our brand identity.

---

## 🌐 Live Website

You can visit the live website here:  
[https://flurbto.github.io/valdex.github.io/)  

> Replace `yourusername` with your GitHub username.

The website features a modern dark theme, smooth animations, and responsive layouts suitable for all devices.

---

## 💡 Mission

Our mission at Valdex is to **bridge creativity and technology** in the gaming world. We focus on:

- Developing mods, tools, and clients for popular games like GTA and Minecraft  
- Showcasing projects in a professional, user-friendly manner  
- Encouraging the gaming and developer communities to experiment, create, and innovate
